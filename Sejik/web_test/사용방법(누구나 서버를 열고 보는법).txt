## 패키지 설치 "IBM_DoctorAI\Sejik\패키지 한번에 설치" 에서 설치할것 (install.ipynb 실행만 하면 됨)

2. conda activate ibm
3. cd 치고 스페이스 치고 backend폴더를 터미널로 드래그
4. python main.py 치고 엔터
5.
